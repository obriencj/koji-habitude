# Release 0.9.0

## Features

- Initial release of koji-habitude
- CLI interface for managing koji objects
- YAML-based configuration with Jinja2 template support
- Support for core koji object types (tags, targets, users, groups, hosts, channels, permissions, external repos, content generators, build types, archive types)
- Automatic dependency resolution and tiered execution
- Template expansion system
- Remote state fetching and comparison
- Change tracking and dry-run support
